## 🎯 About Us

BONJIN은 다양한 기술 스택을 활용하여 혁신적인 애플리케이션을 개발하는 풀스택 개발팀입니다.  
모바일 앱부터 백엔드 서버, 웹 애플리케이션까지 다양한 프로젝트를 진행하고 있습니다.
