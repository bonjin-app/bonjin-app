## 🎯 About Us

**BONJIN**은 모바일부터 백엔드, 웹까지 다양한 기술 스택을 활용하여 혁신적인 애플리케이션을 개발하는 풀스택 개발팀입니다.  
우리는 사용자 경험을 최우선으로 고려하며, 최신 기술 트렌드를 반영한 프로젝트들을 진행하고 있습니다.
